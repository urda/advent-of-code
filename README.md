# Advent of Code (2021)

https://adventofcode.com/2021

## Environment and Tools in Use

- git 2.34.1
- iTerm2 3.4.14
- macOS Monterey 12.0.1
- PyCharm 2021.3 (Professional Edition)
- Python 3.10

Use a virtual environment as such:

```bash
mkvirtualenv advent -p $(which python3.10)
```
