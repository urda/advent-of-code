- [`pytest-cov`](https://pypi.org/project/pytest-cov/)



## Environment Setup

Use a virtual environment as such:

```bash
mkvirtualenv advent -p $(which python3.10)
```

Switch to it:

```bash
workon advent
```

And install the development environment:

```bash
pip install -r requirements-dev.txt
```
